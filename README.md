# Mobile Sales Dashboard - Power BI Project

## Project Overview
This project analyzes mobile phone sales data using Power BI. 
The dashboard provides insights into sales performance by city, brand, and payment method.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Dashboard Features
- Total Sales Analysis
- City-wise Sales Performance
- Brand-wise Revenue
- Payment Method Distribution
- Interactive Filters

## Key Insights
- Major revenue generated from top cities
- Certain brands dominate the market
- UPI and card payments are most used
